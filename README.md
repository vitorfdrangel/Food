### Como executar

#### Iniciando o servidor

1. Vá ao diretório "servidor"
2. Execute o seguinte comando

```bash
npm run start
```

#### Acessando o site

1. Navegue ao diretório "frontend"
2. Execute o seguinte comando para iniciar o vite

```bash
npm run dev
```

3. Abra o localhost:5173
4. Utilize o site
